Pipeline:

1. Load `subject_trials.npy` and metadata CSV. Per `extract_nassar_trials.py` schema: each subject has 4 conditions (`cloud_cp_avoid`, `cloud_cp_seek`, `cloud_drift_avoid`, `cloud_drift_seek`).

2. **Map conditions to model contexts:**
   - `cloud_cp_avoid` and `cloud_cp_seek` → `context='changepoint'`
   - `cloud_drift_avoid` and `cloud_drift_seek` → `context='oddball'`
   - Per CONTEXT.md "per-subject x per-condition" — fit ONE MCMC per (subject, condition) where condition is one of CP or OB. Concatenate avoid+seek trial sequences within each context (or fit separately per avoid/seek if researcher discretion warrants — default is concat per CONTEXT.md "per-condition" interpretation; document the choice in the script docstring).

3. **For each (subject_id, condition) cell:**
   a. Extract `bag_positions = outcome[mask]`, `bucket_positions = prediction[mask]` for trials in this condition. Apply existing extract_nassar_trials.py cleaning (NaN exclusion already done upstream).
   b. Skip cell if fewer than 50 trials remain (insufficient for stable MCMC; record `status='SKIPPED_LOW_TRIALS'` in summary CSV with the n_trials count).
   c. Check if `{output_dir}/per_fit/{subject_id}_{condition}.json` exists — if so, skip (resume support).
   d. Call `fit_with_retry(reduced_bayesian_model, {'bag_positions': bag, 'bucket_positions': bucket, 'context': context}, seed=hash(subject_id) % 2**31)`.
   e. `fit_summary = make_fit_summary(mcmc, status=status, attempts=attempts, var_names=['H','LW','UU','sigma_motor','sigma_LR'], subject_id=subject_id, condition=condition, n_trials=int(len(bag)))`.
   f. Write to `{output_dir}/per_fit/{subject_id}_{condition}.json` via `json.dump(to_jsonable(fit_summary), ...)`.
   g. Print one-line progress: `[{i}/{N}] subject={subject_id} cond={condition} status={status} rhat_max={rhat_max:.3f} ess_min={ess_min:.0f} n_div={n_div}`.
