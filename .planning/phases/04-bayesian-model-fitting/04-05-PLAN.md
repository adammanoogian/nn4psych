2. **Forward-backward implementation** in JAX. The key function:
```python
def hmm_forward_backward(
    emissions_log: jnp.ndarray,    # shape (T, K), log-likelihoods per state per trial
    log_prior: jnp.ndarray,         # shape (K,), log initial state distribution
    log_trans: jnp.ndarray,         # shape (K, K), log transition matrix
) -> tuple[jnp.ndarray, jnp.ndarray]:
    """Standard log-space forward-backward.

    Returns
    -------
    log_alpha : (T, K) forward log-probabilities
    log_likelihood : scalar marginal log-likelihood
    """
    def forward_step(carry, e_t):
        log_alpha_prev = carry
        # log_alpha_t[k] = e_t[k] + logsumexp(log_alpha_prev + log_trans[:, k])
        log_alpha_t = e_t + jax.scipy.special.logsumexp(
            log_alpha_prev[:, None] + log_trans, axis=0
        )
        return log_alpha_t, log_alpha_t

    init = log_prior + emissions_log[0]
    _, log_alpha = jax.lax.scan(forward_step, init, emissions_log[1:])
    log_alpha_full = jnp.concatenate([init[None, :], log_alpha], axis=0)
    log_likelihood = jax.scipy.special.logsumexp(log_alpha_full[-1])
    return log_alpha_full, log_likelihood
```

3. **Per-state Reduced Bayesian emissions:** For each trial t and each state k in {CP, OB}, compute the log-likelihood of the observed bucket update under that state's Reduced Bayesian forward model:
```python
def state_emissions_log(
    bag_positions, bucket_positions, params_per_state,
):
    """Compute (T, K=2) log-likelihoods using compute_rbo_forward per state."""
    log_em_cp = rbo_loglik(bag_positions, bucket_positions, params_per_state['cp'], context='changepoint')
    log_em_ob = rbo_loglik(bag_positions, bucket_positions, params_per_state['ob'], context='oddball')
    return jnp.stack([log_em_cp, log_em_ob], axis=-1)  # shape (T, 2)
```
Where `rbo_loglik` is a JAX function returning per-trial log-likelihoods using the existing `compute_rbo_forward` and the Normal likelihood from `reduced_bayesian.py`.

4. **NumPyro model `contextual_hmm_model(bag_positions, bucket_positions)`:**
   - Sample CP-state Reduced Bayesian params: `H_cp ~ Beta(1.5, 8)`, `LW_cp ~ Beta(2, 2)`, ..., 5 params
   - Sample OB-state params analogously: `H_ob ~ Beta(1.5, 8)`, ..., 5 params
   - Sample initial state distribution: `pi ~ Dirichlet(jnp.array([1.0, 1.0]))`
   - Sample transition matrix rows: `trans = numpyro.sample('trans', dist.Dirichlet(jnp.ones((2, 2))))` — sample two Dirichlets via `with numpyro.plate('K', 2): trans_row ~ Dirichlet(jnp.array([1.0, 1.0]))`
   - Compute `emissions_log = state_emissions_log(bag, bucket, {'cp': cp_params, 'ob': ob_params})`
   - Compute marginal log-likelihood via forward-backward, factor: `numpyro.factor('hmm_loglik', log_likelihood)`

5. **Helper `run_chmm_mcmc(bag, bucket, *, num_warmup, num_samples, num_chains, seed)`** mirroring `reduced_bayesian.run_mcmc` but for the CHMM model.

6. **Posterior predictive helper `chmm_posterior_predictive(mcmc, bag, bucket)`** returning the posterior expected state-occupancy probabilities per trial (shape `(T, 2)`) — useful for the visualization in Task 3.
