**Function 1: fit_latent_circuit_ensemble()**

```python
def fit_latent_circuit_ensemble(
    u: np.ndarray,       # (n_trials, T, input_size=7)
    z: np.ndarray,       # (n_trials, T, output_size=3)
    y: np.ndarray,       # (n_trials, T, N=64)
    n_inits: int = 100,
    n_latent: int = 8,
    epochs: int = 500,
    lr: float = 0.02,
    l_y: float = 1.0,
    weight_decay: float = 0.001,
    sigma_rec: float = 0.15,
    device: str = 'cpu',
    verbose: bool = True,
) -> dict:
```

Implementation:
1. Convert u, z, y to torch.Tensor float32, detach (no grad)
2. Extract dimensions: n_trials = u.shape[0], T = u.shape[1], input_size = u.shape[2], output_size = z.shape[2], N = y.shape[2]
3. For each init i in range(n_inits):
   a. Create fresh LatentNet(n=n_latent, N=N, input_size=input_size, n_trials=n_trials, sigma_rec=sigma_rec, output_size=output_size, device=device)
   b. Call latent_net.fit(u_tensor, z_tensor, y_tensor, epochs=epochs, lr=lr, l_y=l_y, weight_decay=weight_decay)
   c. After fitting, compute final nmse_y: `x = latent_net(u_tensor); nmse_y = latent_net.nmse_y(y_tensor, x).item()`
   d. Also compute mse_z: `latent_net.mse_z(x, z_tensor).item()`
   e. Store: nmse_y, mse_z, and the model state_dict (deep copy via `copy.deepcopy(latent_net.state_dict())`)
   f. Print progress: `Init {i+1}/{n_inits}: nmse_y={nmse_y:.4f}, mse_z={mse_z:.4f}`
   g. Delete the LatentNet instance and call torch.cuda.empty_cache() / gc.collect() to free memory between inits

4. Select best init by LOWEST nmse_y (reconstruction loss)
5. Reload the best state_dict into a fresh LatentNet instance

6. Return dict:
   - 'best_model': the LatentNet instance with best weights loaded
   - 'best_nmse_y': float
   - 'best_mse_z': float
   - 'best_init_idx': int
   - 'all_nmse_y': list of floats (for diagnostics)
   - 'all_mse_z': list of floats
   - 'n_inits': int
   - 'n_latent': int

CRITICAL memory note: 16GB RAM, ~7GB free. Store only state_dict copies (not full model instances) during the ensemble loop. Delete each LatentNet after extracting its state_dict and metrics. Call gc.collect() between inits.

Suppress LatentNet's internal print statements during ensemble by redirecting stdout or modifying the vendored fit() to accept a verbose parameter. If modifying latent_net.py, add `verbose=True` param to fit() and wrap the print block in `if verbose:`. This is acceptable since we vendored the file.

**Function 2: validate_latent_circuit()**

```python
def validate_latent_circuit(
    latent_net: LatentNet,
    u: np.ndarray,
    z: np.ndarray,
    y: np.ndarray,
    W_rec: np.ndarray,   # RNN weight_hh_l0 (64, 64)
    W_in: np.ndarray,    # RNN weight_ih_l0 (64, input_dim) — optional, for Q^T W_in check
    invariant_threshold: float = 0.85,
    device: str = 'cpu',
) -> dict:
```

Implementation — three validation checks from the paper:

**Check 1: Invariant subspace (connectivity-level, Eq. 5/14)**
- Get inferred w_rec: `w_rec = latent_net.recurrent_layer.weight.data.detach().numpy()` — shape (n, n)
- Get Q: `Q = latent_net.q.detach().numpy()` — shape (n, N)
- Compute Q^T W_rec Q: `Q_W_Q = Q @ W_rec @ Q.T` — note W_rec is (N, N), Q is (n, N), so Q @ W_rec gives (n, N), then @ Q.T gives (n, n)
- Wait — Q is shape (n, N). Paper Eq. 14: Q^T W_rec Q = w_rec. Q^T is (N, n), W_rec is (N, N), Q is (n, N). So: Q^T @ W_rec @ Q should be... Q^T is (N, n), which doesn't work for Q^T @ W_rec where W_rec is (N, N) -> result is (N, n)...
- CAREFUL with dimensions. Q is (n, N). Q^T is (N, n). W_rec is (N, N).
  - Paper: Q^T W_rec Q = w_rec. This is (N,n)^T @ (N,N)... that's wrong dimensionally.
  - Re-read: Q is (N, n) in the paper (Eq. 1: y = Qx, y is N-dim, x is n-dim). But in latent_net.py code, self.q has shape (n, N) — it's the TRANSPOSED convention! In the code: `y @ self.q.t()` projects y from N-dim to n-dim, meaning q.t() is (N, n), so q is (n, N).
  - So paper's Q (N, n) = code's q.T (N, n). Paper's Q^T = code's q.
  - Paper Eq. 14: Q^T W_rec Q = w_rec → code: q @ W_rec @ q.T = w_rec_inferred
  - q is (n, N), W_rec is (N, N), q.T is (N, n). Result: (n, N) @ (N, N) @ (N, n) = (n, n). Correct!
- Compute: `Q_W_Q = q @ W_rec @ q.T` where q = latent_net.q.detach().numpy() shape (n, N)
- Compute Pearson correlation: `np.corrcoef(Q_W_Q.flatten(), w_rec.flatten())[0, 1]`
- Report: correlation value, pass/soft-fail vs threshold

**Check 2: Activity-level validation — full space (Qx vs y)**
- Run forward: `x = latent_net(u_tensor)` — shape (n_trials, T, n)
- Project to full space: `qx = (x @ latent_net.q).detach().numpy()` — shape (n_trials, T, N)
- Compare with y: compute R-squared = 1 - SS_res/SS_tot
  - SS_res = sum((qx - y)^2)
  - SS_tot = sum((y - y.mean())^2)
- This is the coefficient of determination r^2 (paper reports r^2 = 0.96 for their fits)

**Check 3: Activity-level validation — latent space (Q^T y vs x)**
- Project y to latent space: `q_y = y @ q.T` — shape (n_trials, T, n) where q.T is (N, n)
  - Wait, q is (n, N), q.T is (N, n). y is (n_trials, T, N). y @ q.T gives (n_trials, T, n). Correct.
- Compare with x (latent states): R-squared between q_y and x.detach().numpy()

Return dict:
```python
{
    'invariant_subspace_corr': float,      # Check 1
    'invariant_subspace_pass': bool,       # >= threshold
    'activity_r2_full_space': float,       # Check 2
    'activity_r2_latent_space': float,     # Check 3
    'nmse_y': float,                       # Final reconstruction error
    'nmse_q': float,                       # Variance unexplained by Q subspace
    'mse_z': float,                        # Task output error
    'threshold': invariant_threshold,
}
```

Note on W_rec orientation: PyTorch nn.RNN weight_hh_l0 has shape (hidden_dim, hidden_dim). The RNN update is h_new = tanh(W_ih @ x + W_hh @ h). So W_hh rows are outputs, columns are inputs — same convention as nn.Linear. Use model.rnn.weight_hh_l0.data.detach().numpy() directly.
