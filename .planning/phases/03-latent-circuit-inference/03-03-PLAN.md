Reference paper perturbation methodology:
- Paper Eq. 6: delta_ij -> q_i @ q_j^T (rank-one perturbation in RNN space)
- Paper Eq. 23: w . delta_ij = W . Q @ delta_ij @ Q^T
- Fig. 4: Two perturbation examples — weakening inhibitory context->sensory connections and weakening excitatory sensory->choice connections
- The perturbation is applied to W_rec of the RNN, then the perturbed RNN is re-run on task trials to measure behavioral change

```python
    model: ActorCritic,
    latent_net: LatentNet,

    n_eval_trials: int = 200,
    perturbation_strengths: list = [-0.5, -0.2, 0.0, 0.2, 0.5],
    device: str = 'cpu',
    env_kwargs: dict = None,
) -> dict:
```

Implementation follows paper Fig. 4 methodology:

**Step 1: Extract Q and w_rec from the fitted LatentNet**
```python
q = latent_net.q.detach()          # (n, N) — code convention
w_rec = latent_net.recurrent_layer.weight.data.detach()  # (n, n)
```

**Step 2: Define perturbation targets**

Following the paper's two example perturbations (Fig. 4a,c):

For our 8-node latent circuit with input_size=7, output_size=3:
- Nodes 0-6 receive inputs (input mask identity in top 7x7)
- Nodes 5-7 produce outputs (output mask identity in bottom 3x3)
- Node identities depend on the learned w_in structure, but we can systematically perturb each connection

Define perturbations as (i, j, strength) tuples where delta_ij is a perturbation from node j to node i in the latent circuit.

Rather than guessing which latent nodes correspond to which task variables (which requires inspecting w_in and w_out), implement a SYSTEMATIC approach:
1. Identify the largest-magnitude connections in w_rec (top 5 excitatory, top 5 inhibitory)
2. For each of these 10 connections, apply perturbation at each strength level
3. Measure task accuracy before and after

**Step 3: Apply rank-one perturbation to RNN (paper Eq. 6/23)**

For each perturbation (i, j, strength):
```python
# Create rank-one perturbation in latent space
delta_ij = torch.zeros(n, n)
delta_ij[i, j] = strength

# Map to RNN space: W_perturbed = W_rec + Q^T @ delta @ Q
# In code convention: q is (n, N), so q^T is (N, n)
# RNN perturbation = q.T @ delta_ij @ q — shape (N, n) @ (n, n) @ (n, N) = (N, N)
W_rec_perturbation = q.T @ delta_ij @ q  # (N, N)

# Apply to model
W_rec_original = model.rnn.weight_hh_l0.data.clone()
model.rnn.weight_hh_l0.data = W_rec_original + W_rec_perturbation
```

**Step 4: Evaluate perturbed model on task**

For each modality_context:
1. Create environment, run n_eval_trials trials
2. Compute task accuracy: fraction of trials where the model's choice at the response period matches the correct answer
3. For context-DM: the correct answer depends on the relevant modality's coherence sign. Positive coherence -> right choice (action 2), negative coherence -> left choice (action 1). Action 0 is fixation.
4. Since we may not have easy access to ground-truth labels from neurogym, use a simpler metric: mean reward per trial (the environment gives reward for correct choices). Compare mean reward between perturbed and unperturbed.

After evaluation, RESTORE original weights:
```python
model.rnn.weight_hh_l0.data = W_rec_original
```

**Step 5: Compile results**

```python
{
    "perturbations": [
        {
            "i": int, "j": int,
            "w_rec_ij": float,  # original latent weight
            "strength": float,
            "baseline_reward": float,
            "perturbed_reward": float,
            "reward_delta": float,
            "baseline_reward_by_context": {0: float, 1: float},
            "perturbed_reward_by_context": {0: float, 1: float},
        },
        ...
    ],
    "summary": {
        "n_perturbations": int,
        "mean_abs_reward_delta": float,
        "max_abs_reward_delta": float,
        "perturbations_with_significant_effect": int,  # |delta| > baseline_std
    }
}
```

Important: The baseline (strength=0.0) should also be evaluated to establish the unperturbed performance level. Use this as the reference for computing deltas.

**Part B: Create scripts/data_pipeline/08_infer_latent_circuits.py**

Create the end-to-end pipeline script that orchestrates the full Phase 3 workflow. Structure:

```python
#!/usr/bin/env python3
"""
08: Latent Circuit Inference Pipeline

Fits a latent circuit model (Langdon & Engel 2025) to context-DM RNN
hidden states and validates the inferred circuit via activity-level,
connectivity-level, and perturbation checks.

Usage:
    # Full pipeline (takes ~2-3 hours)
    python scripts/data_pipeline/08_infer_latent_circuits.py

    # Quick test (5 inits, 50 epochs)
    python scripts/data_pipeline/08_infer_latent_circuits.py --quick

    # Skip data collection (use existing circuit_data.npz)
    python scripts/data_pipeline/08_infer_latent_circuits.py --skip_collection

    # Skip fitting (use existing best_latent_circuit.pt)
    python scripts/data_pipeline/08_infer_latent_circuits.py --skip_fitting
"""

import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).parent.parent.parent))

import argparse
import json
import numpy as np
import torch

from nn4psych.models.actor_critic import ActorCritic
from nn4psych.analysis.circuit_inference import (
    collect_circuit_data,
    save_circuit_data,
    fit_latent_circuit_ensemble,
    validate_latent_circuit,
    perturb_and_evaluate,
)
from nn4psych.analysis.latent_net import LatentNet
from nn4psych.training.resources import configure_cpu_threads
from envs.neurogym_wrapper import SingleContextDecisionMakingWrapper


def main():
    configure_cpu_threads()

    parser = argparse.ArgumentParser(description="Latent circuit inference pipeline")

    # Model args
    parser.add_argument('--model_path', type=str,
                        default='data/processed/rnn_behav/model_context_dm_dual.pth')
    parser.add_argument('--hidden_dim', type=int, default=64)
    parser.add_argument('--input_dim', type=int, default=7)
    parser.add_argument('--action_dim', type=int, default=3)

    # Data collection args
    parser.add_argument('--n_trials_per_context', type=int, default=300)
    parser.add_argument('--data_path', type=str,
                        default='data/processed/rnn_behav/circuit_data.npz')

    # Fitting args
    parser.add_argument('--n_inits', type=int, default=100)
    parser.add_argument('--n_latent', type=int, default=8)
    parser.add_argument('--epochs', type=int, default=500)
    parser.add_argument('--lr', type=float, default=0.02)
    parser.add_argument('--l_y', type=float, default=1.0)
    parser.add_argument('--weight_decay', type=float, default=0.001)

    # Perturbation args
    parser.add_argument('--n_eval_trials', type=int, default=200)

    # Validation args
    parser.add_argument('--invariant_threshold', type=float, default=0.85)

    # Output
    parser.add_argument('--output_dir', type=str, default='output/circuit_analysis')

    # Flags
    parser.add_argument('--quick', action='store_true',
                        help='Quick test: 5 inits, 50 epochs, 50 trials')
    parser.add_argument('--skip_collection', action='store_true',
                        help='Skip data collection, use existing circuit_data.npz')
    parser.add_argument('--skip_fitting', action='store_true',
                        help='Skip fitting, use existing best_latent_circuit.pt')

    args = parser.parse_args()

    if args.quick:
        args.n_inits = 5
        args.epochs = 50
        args.n_trials_per_context = 50
        args.n_eval_trials = 20

    output_dir = Path(args.output_dir)
    output_dir.mkdir(parents=True, exist_ok=True)

    # Step 1: Load model
    model = ActorCritic(args.input_dim, args.hidden_dim, args.action_dim)
    model.load_state_dict(torch.load(args.model_path, map_location='cpu'))
    print(f"Loaded model from {args.model_path}")

    # Step 2: Collect circuit data (or load existing)
    if not args.skip_collection:
        data = collect_circuit_data(
            model, SingleContextDecisionMakingWrapper,
            modality_contexts=[0, 1],
            n_trials_per_context=args.n_trials_per_context,
        )
        save_circuit_data(data, str(Path(args.data_path).parent))
    data_npz = np.load(args.data_path)
    u, z, y = data_npz['u'], data_npz['z'], data_npz['y']
    print(f"Circuit data: u={u.shape}, z={z.shape}, y={y.shape}")

    # Step 3: Fit ensemble (or load existing)
    if not args.skip_fitting:
        result = fit_latent_circuit_ensemble(
            u, z, y,
            n_inits=args.n_inits,
            n_latent=args.n_latent,
            epochs=args.epochs,
            lr=args.lr,
            l_y=args.l_y,
            weight_decay=args.weight_decay,
        )
        torch.save(result['best_model'].state_dict(),
                    output_dir / 'best_latent_circuit.pt')
        # Save diagnostics
        with open(output_dir / 'ensemble_diagnostics.json', 'w') as f:
            json.dump({
                'all_nmse_y': result['all_nmse_y'],
                'all_mse_z': result['all_mse_z'],
                'best_init_idx': result['best_init_idx'],
                'best_nmse_y': result['best_nmse_y'],
            }, f, indent=2)
        best_model = result['best_model']
    else:
        best_model = LatentNet(
            n=args.n_latent, N=args.hidden_dim,
            input_size=u.shape[2], n_trials=u.shape[0],
            output_size=z.shape[2],
        )
        best_model.load_state_dict(
            torch.load(output_dir / 'best_latent_circuit.pt', map_location='cpu'))
        best_model.q = best_model.cayley_transform(best_model.a)

    # Step 4: Validate
    W_rec = model.rnn.weight_hh_l0.data.detach().numpy()
    W_in = model.rnn.weight_ih_l0.data.detach().numpy()
    val = validate_latent_circuit(
        best_model, u, z, y, W_rec, W_in,
        invariant_threshold=args.invariant_threshold,
    )
    # Save validation results
    val_report = {**val, 'data_shape': {'n_trials': int(u.shape[0]),
                                         'T': int(u.shape[1]),
                                         'N': int(y.shape[2])}}
    with open(output_dir / 'validation_results.json', 'w') as f:
        json.dump(val_report, f, indent=2)
    print(f"Invariant subspace corr: {val['invariant_subspace_corr']:.4f}")
    print(f"Activity R2 (full): {val['activity_r2_full_space']:.4f}")

    # Step 5: Perturbation analysis
    pert = perturb_and_evaluate(
        model, best_model,
        SingleContextDecisionMakingWrapper,
        n_eval_trials=args.n_eval_trials,
    )
    with open(output_dir / 'perturbation_results.json', 'w') as f:
        json.dump(pert, f, indent=2)
    print(f"Perturbations tested: {len(pert['perturbations'])}")
    print(f"Mean |reward delta|: {pert['summary']['mean_abs_reward_delta']:.4f}")

    print(f"\nAll results saved to {output_dir}/")
    print("Pipeline complete.")


if __name__ == '__main__':
    main()
```
